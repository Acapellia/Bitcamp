# readme

<내용>
일정간격으로 (오른쪽, 랜덤 높이)에 발판이 생성되고 발판위에는 3위치중 랜덤하게 장애물이 생성된다.
플레이어는 마우스를 통해 점프(2단까지 가능)를 해서 장애물을 피해 발판을 밟아 점수를 올리는 게임

<학습 내용>
- Object Pool을 만들어 매번 발판을 생성하고 파괴하는 것이 아닌 Pool에서 필요할 때마다 빌리고 반납하는 방식
- Pool에 생성 했을 때는 게임 화면에 보이지 않도록 하는 법 
- 배경과 발판을 일정시간에 따라 왼쪽으로 이동시키기
- 하나의 배경을 일정시간에 따라 이어 붙여서 연속된 배경을 만들기
