# readme

<내용>
플레이어는 체력을 가지고 있고 총을 쏴서 좀비를 죽인다.
좀비는 일정시간마다 정해진 spawn장소에서 생성되며 플레이어에게 접근한다.
좀비는 체력과 공격력을 가지고 있으며 플레이어를 공격한다.
일정시간마다 아이템(체력회복, 총알, 코인)이 생성되며 5초 후 사라진다.
